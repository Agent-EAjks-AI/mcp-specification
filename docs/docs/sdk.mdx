---

The Model Context Protocol provides official SDKs for multiple programming languages, making it easy to build MCP servers and clients in your preferred language. Each SDK is actively maintained and follows consistent design patterns across languages.

## Available SDKs

### TypeScript SDK

The official TypeScript SDK for building MCP servers and clients in JavaScript and TypeScript environments.

- **Maintained by**: Anthropic
- **Package**: [`@modelcontextprotocol/sdk`](https://www.npmjs.com/package/@modelcontextprotocol/sdk)
- **Repository**: [modelcontextprotocol/typescript-sdk](https://github.com/modelcontextprotocol/typescript-sdk)
- **Requirements**: Node.js v18.x or higher

### Python SDK

The official Python SDK for building MCP servers and clients in Python.

- **Maintained by**: Anthropic
- **Package**: [`mcp`](https://pypi.org/project/mcp/)
- **Repository**: [modelcontextprotocol/python-sdk](https://github.com/modelcontextprotocol/python-sdk)

### Java SDK

The official Java SDK for building MCP servers and clients in Java.

- **Maintained by**: Spring AI
- **Package**: [`io.modelcontextprotocol.sdk:mcp`](https://mvnrepository.com/artifact/io.modelcontextprotocol.sdk/mcp)
- **Repository**: [modelcontextprotocol/java-sdk](https://github.com/modelcontextprotocol/java-sdk)

### Kotlin SDK

The official Kotlin SDK supporting multiple platforms including JVM, WebAssembly, and iOS.

- **Maintained by**: JetBrains
- **Package**: [`io.modelcontextprotocol:kotlin-sdk`](https://mvnrepository.com/artifact/io.modelcontextprotocol/kotlin-sdk)
- **Repository**: [modelcontextprotocol/kotlin-sdk](https://github.com/modelcontextprotocol/kotlin-sdk)

### C# SDK

The official C# SDK for building MCP servers and clients in .NET environments.

- **Maintained by**: Microsoft
- **Package**: [`ModelContextProtocol`](https://www.nuget.org/packages/ModelContextProtocol)
- **Repository**: [modelcontextprotocol/csharp-sdk](https://github.com/modelcontextprotocol/csharp-sdk)

### Ruby SDK

The official Ruby SDK for building MCP servers and clients in Ruby.

- **Maintained by**: Shopify
- **Package**: [`mcp`](https://rubygems.org/gems/mcp)
- **Repository**: [modelcontextprotocol/ruby-sdk](https://github.com/modelcontextprotocol/ruby-sdk)
