# Tutorials

Learn MCP through hands-on, step-by-step guides. These tutorials will take you from basic concepts to building real-world integrations.

## Getting Started Tutorials

### [Connect MCP Server to Client](/docs/tutorials/connect-server-client)
Learn how to connect existing MCP servers to AI clients like Claude Desktop. Perfect for users who want to enhance their AI assistant with new capabilities.

**What you'll learn:**
- How to configure MCP servers in Claude Desktop
- Testing and troubleshooting connections
- Managing multiple servers

**Prerequisites:** Basic familiarity with JSON configuration files

---

## Building MCP Servers

### [Build a Weather Server](/docs/tutorials/weather-server)
Create a simple MCP server that fetches weather data from an external API. This tutorial introduces core server concepts through a practical example.

**What you'll build:**
- A weather API server with tool calls
- Error handling and validation
- Basic authentication with API keys

**Prerequisites:** Basic Python or TypeScript knowledge

### [Build a System Theme Server](/docs/tutorials/theme-server)
Build a local MCP server that can change your operating system's theme (light/dark mode). Learn about local system integration and security considerations.

**What you'll build:**
- Local system integration server
- Platform-specific implementations
- Permission and security handling

**Prerequisites:** Python or TypeScript, basic system administration

### [Build a Google Calendar Server](/docs/tutorials/gcal-server)
Create a remote MCP server that integrates with Google Calendar API. Covers OAuth authentication, cloud deployment, and real-world API integration.

**What you'll build:**
- OAuth 2.0 authenticated server
- Google Calendar API integration
- Cloud deployment setup

**Prerequisites:** Understanding of OAuth flows, cloud services basics

---

## Building MCP Clients

### [Build a Node.js Client](/docs/tutorials/building-a-client-node)
Create a custom MCP client application using Node.js. Learn how to integrate MCP capabilities into your own applications.
