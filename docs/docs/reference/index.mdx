## Quick Reference

### Common Message Types

| Message Type | Direction | Purpose |
|--------------|-----------|---------|
| `initialize` | Client → Server | Initialize connection |
| `tools/list` | Client → Server | List available tools |
| `tools/call` | Client → Server | Execute a tool |
| `resources/list` | Client → Server | List available resources |
| `resources/read` | Client → Server | Read resource content |
| `prompts/list` | Client → Server | List available prompts |
| `prompts/get` | Client → Server | Get prompt template |
| `sampling/createMessage` | Server → Client | Request LLM completion |

### Standard Error Codes

| Code | Description |
|------|-------------|
| -32700 | Parse error |
| -32600 | Invalid request |
| -32601 | Method not found |
| -32602 | Invalid params |
| -32603 | Internal error |
| -32000 | Server error |

### Transport Comparison

| Transport | Use Case | Security | Performance |
|-----------|----------|----------|-------------|
| stdio | Local servers | Process isolation | High |
| HTTP/HTTPS | Remote servers | TLS encryption | Medium |
| WebSocket | Real-time apps | TLS encryption | High |
